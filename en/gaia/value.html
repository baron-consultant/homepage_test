GAIA Features

Real-time collaboration and visualized UX
for efficient field-data management,
with instant mobile data sharing

Effective
Information Management

We ensure the security and reliability of on-site information by designating real workers for each borehole and storing and managing stratigraphic data by creating web records, and share the on-site situation in real time through monitoring of on-site work conditions to support collaboration between (site) workers and (headquarters) managers.

User-Centered
Design

Visualizes layer symbols, colors, sample types, moisture conditions, etc. Supports easy data entry with a WYSIWYG design

Mobile Device
Support

Using smartphones and tablets, users can process borehole data, geological layer information, groundwater levels, site conditions, and sample-box information in real time and share various field information instantly.
