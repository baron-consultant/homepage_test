가주상도 자동 생성
Web 서버를 활용하여 현장에서 기록지를 작성함과 동시에 실시간으로 시추조사 결과를 반영한 가주상도가 자동 생성되며, 이는 현장기록지와의 비교 검토를 통하여 시추정보의 신뢰성을 제고합니다.

WYSIWYG 방식의 입력 지원
WYSIWYG(What You See Is What You Get, “보는대로 얻는다”)방식의 사용자 친화적인 인터페이스를 통해 지층별 심볼, 색상, 시료형태, 함수상태 등 상세 입력 사항을 시각화하고 간편한 입력이 가능합니다.
사용자 친화적인
인터페이스
  • 간편한 입력
  • 다양한 입력 사항 시각화
  • 지층 심벌, 색조, 시료형태
